Converters¶

Functions to convert from one format to another.

prose.converters.old_to_new(folder_path, destination=None, keyword_observatory='OBSERVAT')¶

Convert an old data products folder (contianing *.phots and *_stack.fits) into .phot

Parameters

  • folder_path (str) – path of old data products folder

  • destination (str, optional) – path of converted file (must contain extension .phot), by default None and a {folder_path}.phot is created

  • keyword_observatory (str, optional) – fits header where telescope name can be found, by default “OBSERVAT”

prose.converters.AIJ_to_phot(aij_file, telescope, destination, stack)¶

Convert an AIJ data file to a phot file

Parameters

  • aij_file (str) – path of the AIJ file

  • telescope (str) – telescope name

  • destination (str) – path of the destination .phot file (must include filename)

  • stack (str) – path of the corresponding stack image
